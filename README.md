# LinkForge

> Forge Better Links.

LinkForge is an open-source link management platform that allows users to shorten URLs, manage links, generate QR codes, and track analytics through a modern dashboard.

## ✨ Features

- 🔗 URL Shortening
- 📊 Analytics Dashboard
- 🎯 Custom Aliases
- 📱 QR Code Generation
- 🔐 Secure Authentication
- 🌍 Responsive Design

## 🛠 Tech Stack

### Frontend
- React
- Vite
- Tailwind CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB
